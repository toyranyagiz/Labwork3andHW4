# Homework4

This is my GAD2006 Homework4 project made in Unreal Engine.

The project is based on a grid-based tile placement system. It uses the Flyweight pattern idea by keeping shared tile data in tile classes and placing tiles with `UInstancedStaticMeshComponent`.

## Features

- Grid-based tile placement
- Multiple tile types
- Current tile preview on grid selection
- Mouse scroll down to change to the next tile
- Mouse scroll up to change to the previous tile
- Right click to rotate the selected tile by 90 degrees
- Instanced Static Mesh usage

## Controls

- Left Click: Place tile
- Scroll Down: Next tile
- Scroll Up: Previous tile
- Right Click: Rotate tile
